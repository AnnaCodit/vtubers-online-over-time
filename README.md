# VTubers Online Over Time

Этот проект предназначен для сбора и агрегации исторической статистики среднего онлайна (Average Viewers) различных стримеров (витуберов) с сайта [TwitchTracker](https://twitchtracker.com/).

Собранная статистика приводится к единой временной шкале и форматируется в виде широкой таблицы, идеально подходящей для создания анимаций графиков (например, Bar Chart Race).

## Быстрый запуск

1. Установите зависимости:
   ```bash
   pip install requests
   ```
2. Поместите никнеймы стримеров в файл `vtubers.txt` (каждый никнейм на новой строке).
3. Запустите сборщик сырых данных:
   ```bash
   python scrape_all_vtubers.py
   ```
   *Скрипт создаст файл `data/vtubers_avg_online.csv`.*
4. Преобразуйте данные в итоговый формат с добавлением аватарок:
   ```bash
   python create_wide_dataset.py
   ```
   *Скрипт создаст файл `data/vtubers_wide_stats.csv`.*

Подробное описание работы скриптов и обхода ограничений TwitchTracker находится в файле [DOCS.md](DOCS.md).
